ElasticSearch-Query

Query and Filter Context

####Query
query字句表示”该文档与此查询字句的匹配程度如何？”。出了确定文档是否匹配之外，query字句还计算一个_score,表示文档相对于其他文档的匹配程度。
只要将查询子句传递给查询参数(例如searchAPI中的查询参数),查询上下文就会生效。

####Filter
filter字句表示”该文档是否与查询字句匹配？”，它不会计算_score，只会回答是或者不是。频繁的使用filter会自动被ElasticSearch缓存，以此来加速查询效率。
只要将查询子句传给filter参数(例如bool查询中的filter或者must_not参数，constant_score中的filter参数或者filter聚合)。
